About

This is a fork of gddo, the software which previously powered the now-defunct godoc.org, operated by Drew DeVault. The source code is available on git.sr.ht.

godoc.io displays documentation for GOOS=linux unless otherwise noted at the bottom of the documentation page.

Add a package to godoc.io

godoc.io generates documentation from Go source code. The guidelines for writing documentation for the godoc tool apply to godoc.io.

It's important to write a good summary of the package in the first sentence of the package comment. godoc.io indexes the first sentence and displays the first sentence in package lists.

To add a package to godoc.io, search for the package by import path. If godoc.io does not already have the documentation for the package, then godoc.io will fetch the source from the version control system on the fly and add the documentation.

godoc.io checks for package updates once per day. You can force godoc.io to update the documentation immediately by clicking the refresh link at the bottom of the package documentation page.

godoc.io crawls package imports and child directories to find new packages.

Remove a package from godoc.io

godoc.io automatically removes packages deleted from the version control system when godoc.io checks for updates to the package. You can force godoc.io to remove a deleted package immediately by clicking the refresh link at the bottom of the package documentation page. If you do not want godocs.io to display documentation for your package, send mail to sir@cmpwn.com with the import path of the path of the package that you want to remove.

Keyboard Shortcuts

godocs.io has keyboard shortcuts for navigating package documentation pages. Type '?' on a package page for help. {{end}}
