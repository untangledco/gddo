About

This is a fork of gddo, the software which previously powered the now-defunct godoc.org{{with config.AdminName}}, operated by {{.}}{{end}}. The source code is available on sr.ht.

Documentation

{{config.BrandName}} generates documentation from Go source code. The guidelines for writing documentation for the godoc tool apply.

It's important to write a good summary of the package in the first sentence of the package comment. {{config.BrandName}} indexes the first sentence and displays it in search results.

Add a package

To add a package to {{config.BrandName}}, search for the package by import path. If {{config.BrandName}} does not already have the documentation for the package, it will fetch the package source code on the fly and add the documentation.

{{config.BrandName}} checks for package updates periodically. You can force {{config.BrandName}} to update the documentation immediately by clicking the refresh button at the bottom of the package documentation page.

Remove a package

If you do not want {{config.BrandName}} to display documentation for your package, send mail to {{config.AdminEmail}} with the import path of the path of the package that you want to remove. {{- end}}

Keyboard Shortcuts

{{config.BrandName}} has keyboard shortcuts for navigating package documentation pages. Type '?' on a package page for help. {{end}}
